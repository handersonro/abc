a) Para acessar a opção de pesquisa de reunião, o usuário deve acionar o item “Reunião” e subitem “Pesquisar reunião” conforme imagem a seguir:

b) Então, o sistema exibe a tela contendo os campos para preenchimento das informações:

A pesquisa poderá ser realizada de duas formas:
Pesquisa por campos;
Pesquisa geral.

c) Pesquisa por campos: o usuário preenche um ou mais campos com as informações desejadas e aciona o botão PESQUISAR. O sistema realiza busca por registros que contenham os dados informados e os apresenta na tela. Na ausência de resultados de pesquisa o sistema informará que não existem linhas para exibir.

d) Pesquisa geral: o usuário não preenche nenhum campo e aciona o botão PESQUISAR. O sistema realiza busca e exibe as reuniões cadastradas no sistema para a autoridade competente. Na ausência de resultados de pesquisa o sistema informará que não existem linhas para exibir.
