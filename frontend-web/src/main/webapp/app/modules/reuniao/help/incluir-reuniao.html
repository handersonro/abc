b) Ao acionar o subitem “Incluir reunião” o sistema apresenta o formulário para cadastro conforme imagem a seguir:

c) Devem ser informados obrigatoriamente os campos que contém * (asterisco) em seu nome.

d) Os campos Data inicial e Data final e Data de cadastro abrem o calendário para seleção da data e hora de realização da reunião e este último para alteração da data de cadastro da reunião.

e) A seleção da data e hora é realizada em três etapas, conforme exemplificado nas imagens abaixo. Na primeira etapa, o usuário seleciona o ano, mês e dia desejados e seleciona o botão OK. Assim, o sistema abre a tela para seleção da hora. O usuário seleciona a hora desejada e novamente clica em botão OK. Na última etapa o sistema habilita o cursor de minutos. O usuário seleciona o dado e clica no botão OK. O sistema fecha o calendário e carrega a informação na tela Incluir reunião.

f) No campo Vincular participante deve ser informado o nome de cada um dos participantes da reunião. A partir da terceira letra digitada o sistema traz a lista de pessoas da rede interna do Mtur que contém em seu nome as letras fornecidas, conforme imagem abaixo. O usuário deve selecionar o registro desejado.

h) Os campos Assunto e Pauta da reunião contém uma paleta de edição de conteúdo na qual o usuário poderá escolher qual formatação da fonte digitada no campo.

j) O segundo grupo de ícones de formatação são para o usuário definir a cor, o alinhamento do texto e também poderá adicionar links e imagens no campo. Para definir a cor da fonte o usuário deve selecionar os dados digitados e acionar o ícone de Font color conforme exemplificado na imagem abaixo:

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela Incluir reunião.

k) Após o preenchimento dos campos o usuário deve acionar o botão SALVAR. O novo registro será armazenado no sistema.

Ao salvar uma reunião, um e-mail de convocação é encaminhado aos participantes. Neste e-mail conterá links que permitem aceitar ou recusar o convite. Por meio da funcionalidade Emitir Lista, o usuário pode consultar os participantes que aceitaram ou recusaram a convocação.

l) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

m) Para abortar a operação de cadastro de reunião, o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
