a) Para acessar a opção de editar reunião, o usuário deve acionar o ícone “Ações” localizado na tela Pesquisar reunião conforme exemplificado na imagem abaixo:

b) Acionado o ícone Ações, o sistema disponibiliza as opções Editar, Remover e Emitir lista. O usuário deve selecionar a ação Editar:

c) O sistema mostra os dados da reunião para edição:

d) Os campos Data inicial e Data final e Data de cadastro abrem o calendário para alteração das datas, caso o usuário deseje.

e) A seleção da data e hora é realizada em três etapas, conforme exemplificado nas imagens abaixo. Na primeira etapa, o usuário seleciona o ano, mês e dia desejados e seleciona o botão OK. Assim, o sistema abre a tela para seleção da hora. O usuário seleciona a hora desejada e novamente clica em botão OK. Na última etapa o sistema habilita o cursor de minutos. O usuário seleciona o dado e clica no botão OK. O sistema fecha o calendário e carrega a informação na tela Editar reunião.

f) Para incluir um novo participante na reunião, no campo Vincular participante o usuário deve digitar o nome desejado. A partir da terceira letra digitada o sistema traz a lista de pessoas da rede interna do Mtur que contém em seu nome as letras fornecidas. O usuário deve selecionar o registro desejado.

g) Se desejar retirar um participante da lista o usuário deve clicar sobre o ícone “X” localizado ao lado do nome na tela Incluir reunião.

h) Os campos Assunto e Pauta da reunião contém uma paleta de edição do conteúdo na qual o usuário poderá escolher qual formatação da fonte digitada no campo.

i) O primeiro grupo de ícones de formatação da paleta de edição, são para o usuário definir o tamanho da fonte das informações digitadas no campo, conforme imagem abaixo:

j) O segundo grupo de ícones de formatação são para o usuário definir a cor, o alinhamento do texto e também poderá adicionar links e imagens no campo. Para definir a cor da fonte o usuário deve selecionar os dados digitados e acionar o ícone de Font color conforme exemplificado na imagem abaixo:

k) O usuário deve alterar os dados desejados e acionar o botão SALVAR. A alteração será armazenada no sistema.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela Incluir reunião.

l) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

m) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava a alteração e retorna para a tela inicial.
