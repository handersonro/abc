b) Acionado o ícone Ações, o sistema disponibiliza as opções Editar e Remover. O usuário deve selecionar a ação Remover:

c) O sistema apresenta mensagem solicitando a confirmação da exclusão do registro de remetente:

d) Para confirmar a exclusão, acione o botão OK e para abortar, clique no botão CANCELAR. Confirmada a exclusão, o registro é eliminado do sistema.

A exclusão não é permitida para remetentes que estejam associados à um convite ou à uma audiência. Nestes casos, é exibida uma mensagem de alerta.
