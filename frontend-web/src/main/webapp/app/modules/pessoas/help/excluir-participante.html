d) Para confirmar a exclusão, acione o botão OK e para abortar, clique no botão CANCELAR. Confirmada a exclusão, o registro é eliminado do sistema.

A exclusão não é permitida para participantes que estejam associados à uma audiência cadastrada no sistema. Nestes casos, é exibida uma mensagem de alerta.
