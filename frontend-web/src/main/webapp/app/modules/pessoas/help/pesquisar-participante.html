a) Para acessar a opção de pesquisa de participante, o usuário deve acionar o item “Pessoa” e subitem “Pesquisar participante” conforme imagem a seguir:

b) Então, o sistema exibe a tela contendo os campos para preenchimento das informações:

A pesquisa poderá ser realizada de duas formas:
Pesquisa por campos;
Pesquisa geral.

c) Pesquisa por campos: o usuário preenche um ou mais campos com as informações desejadas e aciona o botão PESQUISAR. O sistema realiza busca por registros que contenham os dados informados e os apresenta na tela. Na ausência de resultados de pesquisa o sistema informará que não existem linhas para exibir.

d) Pesquisa geral: o usuário não preenche nenhum campo e aciona o botão PESQUISAR. O sistema realiza busca e exibe os participantes cadastrados no sistema. Na ausência de resultados de pesquisa o sistema informará que não existem linhas para exibir.

Nesta pesquisa será exibido somente os participantes cadastrados na tela Incluir Participante. Os participantes vinculados à uma reunião, são usuários internos (da rede) Mtur e não estão disponíveis para pesquisa e edição de seus registros.
