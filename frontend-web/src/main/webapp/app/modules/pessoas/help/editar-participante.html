a) Para acessar a opção de editar o registro de participante, o usuário deve acionar o ícone “Ações” localizado na tela Pesquisar participante conforme

b) Acionado o ícone Ações, o sistema disponibiliza as opções Editar e Remover. O usuário deve selecionar a ação Editar:

c) O sistema mostra os dados do participante na tela para edição:

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela. O campo e-mail não aceita endereços que não contenham o símbolo @ e o ponto em sua composição.

d) O usuário deve alterar os dados desejados e acionar o botão SALVAR. A alteração será armazenada no sistema.

e) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

f) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava a alteração e retorna para a tela inicial.
