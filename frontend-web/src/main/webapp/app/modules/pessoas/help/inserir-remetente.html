c) Devem ser informados obrigatoriamente os campos que contém * (asterisco) em seu nome.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela. O campo e-mail não aceita endereços que não contenham o símbolo @ em sua composição.

d) Após o preenchimento dos campos o usuário deve acionar o botão SALVAR. O novo registro será armazenado no sistema.

e) Os remetentes cadastrados podem ser pesquisados, alterados e incluídos em convites e audiências de qualquer autoridade.

f) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

g) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
