b) Ao acionar o subitem “Incluir convite” o sistema apresenta o formulário para cadastro conforme imagens a seguir:

c) Os campos identificados com asterisco deverão ser obrigatoriamente preenchidos.

d) Os campos Data inicial, Data final e Data de cadastro abrem o calendário para seleção da data e hora de realização do evento e este último para alteração da data de cadastro do convite.

e) A seleção da data e hora é realizada em três etapas, conforme exemplificado nas imagens abaixo. Na primeira etapa, o usuário seleciona o ano, mês e dia desejados e seleciona o botão OK. Assim, o sistema abre a tela para seleção da hora. O usuário seleciona a hora desejada e novamente clica em botão OK. Na última etapa o sistema habilita o cursor de minutos. O usuário seleciona o dado e clica no botão OK. O sistema fecha o calendário e carrega a informação na tela Incluir convite.

f) O campo remetente pesquisa os registros cadastrados na aplicação “Incluir remetentes” a partir da digitação de três caracteres. Ao visualizar os resultados o usuário deverá selecionar o escolhido. Apenas um remetente é permitido por convite.

g) O usuário deverá informar se o evento é nacional ou internacional. Para o primeiro tipo o campo obrigatório “Local do evento” será apresentado, este campo pesquisa a lista de municípios brasileiros a partir da digitação de três caracteres. Ao visualizar os resultados o usuário deverá selecionar o escolhido. Apenas um município é permitido por convite.

h) Para o evento internacional os campos obrigatórios “País” e “Cidade” serão apresentados. O campo “País” pesquisa uma lista de países a partir da digitação de três caracteres. Ao visualizar os resultados o usuário deverá selecionar o escolhido. Apenas um país é permitido por convite. O campo “Cidade” deverá ser preenchido pelo usuário.

i) Os campos “Assunto” e “Observação” possuem uma paleta de edição de conteúdo no qual o usuário poderá escolher qual formatação da fonte digitada no campo.

j) O primeiro grupo de ícones de formatação da paleta de edição, são para o usuário definir o tamanho da fonte das informações digitadas no campo, conforme imagem abaixo.

k) O segundo grupo de ícones de formatação são para o usuário definir a cor, o alinhamento do texto e também poderá adicionar links e imagens no campo. Para definir a cor da fonte o usuário deve selecionar os dados digitados e acionar o ícone de Font color conforme exemplificado na imagem abaixo.

l) Após o preenchimento dos campos o usuário deve acionar o botão SALVAR. O novo registro será armazenado no sistema.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela.

m) Os convites cadastrados podem ser pesquisados, editados ou excluídos do sistema a qualquer tempo.

n) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

o) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
