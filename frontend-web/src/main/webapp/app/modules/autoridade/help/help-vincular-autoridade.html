a) Para acessar a opção de vincular autoridade, o usuário deve acionar o item “Autoridade” e subitem “Vincular autoridade” conforme imagem a seguir:

b) Então, o sistema exibe a tela contendo os campos para preenchimento das informações:

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela.

c) Deve ser preenchido o nome do usuário que será vinculado a autoridade, conforme imagem a seguir:

d) Após o preenchimento do nome do usuário, as opções de autoridade anteriormente cadastradas, serão visualizadas.

e) Após o preenchimento dos campos o usuário deve acionar o botão SALVAR. A vinculação do usuário à autoridade será realizada, exibindo a mensagem: “Registro incluído com sucesso”.

f) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

g) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
