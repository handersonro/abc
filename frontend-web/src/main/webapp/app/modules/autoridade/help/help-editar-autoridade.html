a) Para acessar a opção de editar o registro da autoridade, o usuário deve acionar o ícone “Ações” conforme exemplificado na imagem abaixo:

b) Acionado o ícone “Ações”, o sistema disponibiliza a opção de editar.

c) O sistema exibe na tela os dados da autoridade selecionada para realizar a edição.

O usuário deve alterar os dados desejados e acionar o botão SALVAR. A alteração será armazenada no sistema.

d) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

e) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
