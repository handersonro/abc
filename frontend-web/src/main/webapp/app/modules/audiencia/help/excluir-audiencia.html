a) Para acessar a opção de excluir o registro de audiência, o usuário deve acionar o ícone “Ações” localizado na tela Pesquisar audiência conforme exemplificado na imagem abaixo:

b) Acionado o ícone Ações, o sistema disponibiliza as opções Editar e Remover. O usuário deve selecionar a ação Remover:

c) O sistema apresenta mensagem solicitando a confirmação da exclusão do registro de audiência:

d) Para confirmar a exclusão, acione o botão OK. Confirmada a exclusão, o registro é eliminado do sistema.

e) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
