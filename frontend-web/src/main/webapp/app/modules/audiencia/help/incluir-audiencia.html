c) Devem ser informados obrigatoriamente que contém * (asterisco) em seu nome.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela. O campo e-mail não aceita endereços que não contenham o símbolo @ em sua composição.

d) Os campos Data inicial e Data final e Data de cadastro abrem o calendário para seleção da data e hora de realização da audiência e este último para alteração da data de cadastro da audiência.

e) A seleção da data e hora é realizada em três etapas, conforme exemplificado nas imagens abaixo. Na primeira etapa, o usuário seleciona o ano, mês e dia desejados e seleciona o botão OK. Assim, o sistema abre a tela para seleção da hora. O usuário seleciona a hora desejada e novamente clica em botão OK. Na última etapa o sistema habilita o cursor de minutos. O usuário seleciona o dado e clica no botão OK. O sistema fecha o calendário e carrega a informação na tela Incluir audiência.

f) No campo Vincular participante deve ser informado o nome de cada um dos participantes da audiência. A partir da terceira letra digitada o sistema traz a lista dos participantes cadastrados que contém em seu nome as letras fornecidas, conforme imagem abaixo. O usuário deve selecionar o registro desejado.

g) Se desejar retirar um participante da lista o usuário deve clicar sobre o ícone “X” localizado ao lado do nome na tela Incluir audiência.

h) No campo Local da Audiência deve ser informado o local onde acontecerá a audiência. A partir da terceira letra digitada o sistema traz a lista dos Municípios que contém em seu nome as letras fornecidas, conforme imagem abaixo. O usuário deve selecionar o registro desejado.

i) Os campos Assunto e Observação possuem uma paleta de edição de conteúdo na qual o usuário poderá escolher qual formatação da fonte digitada no campo.

j) O primeiro grupo de ícones de formatação da paleta de edição, são para o usuário definir o tamanho da fonte das informações digitadas no campo, conforme imagem abaixo.

k) O segundo grupo de ícones de formatação são para o usuário definir a cor, o alinhamento do texto e também poderá adicionar links e imagens no campo. Para definir a cor da fonte o usuário deve selecionar os dados digitados e acionar o ícone de Font color conforme exemplificado na imagem abaixo.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela Incluir audiência.

l) Após o preenchimento dos campos o usuário deve acionar o botão SALVAR. O novo registro será armazenado no sistema.

m) Para limpar os campos já preenchidos o usuário deve clicar sobre o botão LIMPAR.

n) Para abortar a operação o usuário deve acionar o botão CANCELAR. Neste caso, o sistema não grava o registro e retorna para a tela inicial.
