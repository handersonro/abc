a) Para acessar a opção de editar o registro de audiência, o usuário deve acionar o ícone “Ações” localizado na tela Pesquisar audiência conforme exemplificado na imagem abaixo:

b) Acionado o ícone Ações, o sistema disponibiliza as opções Editar e Remover. O usuário deve selecionar a ação Editar:

c) O sistema mostra os dados do registro de audiência na tela para edição:

d) No campo Vincular participante deve ser informado o nome de cada um dos participantes da audiência. A partir da terceira letra digitada o sistema traz a lista dos participantes cadastrados que contém em seu nome as letras fornecidas, conforme imagem abaixo. O usuário deve selecionar o registro desejado.

e) Se desejar retirar um participante da lista o usuário deve clicar sobre o ícone “X” localizado ao lado do nome na tela Editar audiência.

f) No campo Local da Audiência deve ser informado o local onde acontecerá a audiência. A partir da terceira letra digitada o sistema traz a lista dos Municípios que contém em seu nome as letras fornecidas, conforme imagem abaixo. O usuário deve selecionar o registro desejado.

g) Os campos Assunto e Observação possuem uma paleta de edição de conteúdo na qual o usuário poderá escolher qual formatação da fonte digitada no campo.

h) O primeiro grupo de ícones de formatação da paleta de edição, são para o usuário definir o tamanho da fonte das informações digitadas no campo, conforme imagem abaixo.

i) O segundo grupo de ícones de formatação são para o usuário definir a cor, o alinhamento do texto e também poderá adicionar links e imagens no campo. Para definir a cor da fonte o usuário deve selecionar os dados digitados e acionar o ícone de Font color conforme exemplificado na imagem abaixo.

O botão SALVAR é habilitado somente quando o usuário preencher corretamente todos os campos obrigatórios da tela Editar audiência.
